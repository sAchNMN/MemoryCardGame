# 记忆翻牌大师 - 纯代码版

微信小游戏，只有一个 `game.js` 搞定全部。

## 使用步骤

1. **打开微信开发者工具**
2. **新建小游戏项目**
3. AppID 填入你的小程序 AppID
4. 项目目录选择 `MemoryCardGame-Pure/`
5. 点击「编译」

## 项目结构

```
MemoryCardGame-Pure/
├── game.js               ← 全部游戏代码（唯一核心文件）
├── game.json             ← 游戏配置
└── project.config.json   ← 项目配置
```

## 文件说明

- **`game.js`** — 包含：Canvas 渲染、卡片逻辑、游戏流程、计时计分、微信分享/排行榜、全部 UI
- 没有第三方依赖，没有编辑器，没有场景文件
- 所有卡片图案用 Canvas 2D API 绘制（星/心/钻/叶/月/日/闪/雪）

## 游戏玩法

1. 开局所有卡片展示 1 秒（记忆阶段）
2. 全部翻回背面
3. 点击翻两张，配对成功加分，失败翻回
4. 5 分钟倒计时，全部配对或时间到则结束
5. 自动上传分数到好友排行榜
