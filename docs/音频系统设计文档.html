6. 技术实现方案

6.1 程序化合成 (Procedural Audio)

所有音效通过 Web Audio API 实时合成，零外部依赖：

// 核心合成引擎示例：翻牌音效
AudioManager.prototype.playFlip = function() {
  const ctx = this.audioCtx;
  const osc = ctx.createOscillator();
  const gain = ctx.createGain();
  osc.type = 'sine';
  osc.frequency.setValueAtTime(400, ctx.currentTime);
  osc.frequency.exponentialRampToValueAtTime(800, ctx.currentTime + 0.08);
  gain.gain.setValueAtTime(0.3, ctx.currentTime);
  gain.gain.exponentialRampToValueAtTime(0.001, ctx.currentTime + 0.08);
  osc.connect(gain).connect(ctx.destination);
  osc.start(); osc.stop(ctx.currentTime + 0.08);
};

6.2 自适应 BGM 选段

使用 Web Audio Oscillator 序列 生成短旋律循环，通过 setState(state) 切换：

// BGM 状态切换
audio.setState('MENU');    // → 安静钢琴旋律
audio.setState('PLAYING'); // → 轻快完整编曲
audio.setState('OVER');    // → 胜利/失败和弦

6.3 微信小游戏集成

微信小游戏支持 wx.createWebAudioContext()（基础库 2.28.0+）。底层封装：

// 惰性初始化 — 用户首次交互后才创建 AudioContext
var AudioManager = {
  _ctx: null,
  _ready: false,
  init: function() {
    try {
      this._ctx = wx.createWebAudioContext();
      this._ready = true;
    } catch(e) {
      console.warn('[Audio] WebAudio not supported, audio disabled');
    }
  }
};

7. 集成计划
